# Netflix Data Visualization using Matplotlib

This is a small practice project where I explored a Netflix dataset and tried to understand the data using Python.

Instead of looking at large tables of numbers, I used graphs and charts to make the data easier to understand. Data visualization helps in quickly seeing patterns, trends and comparisons in the data.

## Tools Used

Python
Pandas
Matplotlib
Google Colab

## What this project shows

In this project I created different types of charts such as:

* Bar chart to compare number of Movies and TV Shows
* Pie chart to see the distribution of content ratings
* Histogram to understand movie duration distribution
* Scatter plot to see the relationship between release year and number of shows
* Bar chart to find which countries produce the most Netflix content

## Files in this repository

Netflix_data_visualisation_using_Matplotlib.ipynb – main notebook with Netflix dataset analysis
matplotlib_basics.ipynb – practice notebook for learning different matplotlib charts
Netflix_Dataset.csv – dataset used in the project
